# JobConnect Frontend

React frontend for JobConnect using React Router, Axios, Bootstrap, and Vite.

## Setup

```bash
cp .env.example .env
npm install
npm run dev
```

The default API URL is configured in `.env.example`:

```env
VITE_API_URL=http://localhost:5000/api
```

The app runs at:

```text
http://localhost:5173
```

Run a production build with:

```bash
npm run build
```
